# Unemployment Analysis with Python

## 📌 Project Overview
This project analyzes unemployment rate data using Python.  
The dataset represents the percentage of unemployed people across different regions and time periods.

The goal of this project is to clean the data, explore unemployment trends, study the impact of Covid-19, identify important patterns, and generate insights useful for economic planning.

---

## 🎯 Objectives

- Analyze unemployment rate data.
- Perform data cleaning and preprocessing.
- Visualize unemployment trends using charts.
- Investigate the impact of Covid-19 on unemployment rates.
- Identify monthly and regional patterns.
- Present useful insights for policy making.

---

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## 📂 Dataset Features

- Region
- Date
- Frequency
- Estimated Unemployment Rate (%)
- Estimated Employed
- Estimated Labour Participation Rate (%)

---

## 📊 Analysis Performed

### ✅ Data Cleaning
- Removed unnecessary spaces in column names
- Converted date column into datetime format
- Checked missing values
- Created Month and Year columns

### ✅ Exploratory Data Analysis
- Mean, Maximum, Minimum unemployment rate
- Top 5 highest unemployment cases
- Region-wise unemployment comparison
- Monthly average unemployment rate trends

### ✅ Visualization
- Line chart for unemployment rate over time
- Bar chart for average unemployment rate by region
- Line chart for monthly average unemployment rate
- A multi-line chart to show how unemployment rates varied across states over time
- Heatmap for correlation

---

## 🦠 Covid-19 Impact

The analysis showed a rise in unemployment during lockdown months.  
Several states recorded unusually high unemployment rates during April and May 2020.

---

## 📈 Key Insights

- Mean unemployment rate was around 12.24%.
- Maximum unemployment reached 75.85%.
- Puducherry recorded the highest unemployment in April 2020.
- Employment and unemployment showed negative correlation.
- South and East regions were highly affected.
